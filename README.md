🥊 HIIT Engine
A fast, offline‑ready interval trainer built for focus, discipline, and daily momentum.
The HIIT Engine is a lightweight, privacy‑first workout timer designed for quick sessions, structured intervals, and zero distractions. It runs entirely in the browser, saves your settings locally, and works even without internet.

🚀 Features
- Customizable work, rest, and round durations
- Clean, distraction‑free interface
- Audio cues for transitions
- Real‑time progress display
- Session summary screen
- Automatic LocalStorage saving
- Fully offline via Service Worker
- No accounts, no cloud, no tracking

🧠 How It Works
The HIIT Engine uses a simple state machine to move between:
- Work phase
- Rest phase
- Round transitions
- Session completion
All user preferences (intervals, rounds, last session) are stored locally on the device. Nothing is uploaded or shared.

📦 Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage
- Service Worker (PWA support)
